# CV / LinkedIn Project Description

## Short Version
**Daily Revenue Forecasting — VinUniversity Datathon 2026**  
Built a machine learning pipeline to forecast daily revenue for a simulated Vietnamese fashion e-commerce business. Engineered calendar, holiday, cyclical, lag, and rolling-window features; tuned XGBoost, LightGBM, CatBoost, and Random Forest models; and achieved the best notebook experiment result with CatBoost, reaching MAE of 71.4K, RMSE of 99.7K, and R² of 0.9960.

## Strong CV Bullet Points
- Developed an end-to-end daily revenue forecasting pipeline for VinUniversity Datathon 2026 using Python, Pandas, Scikit-learn, XGBoost, LightGBM, CatBoost, and Random Forest.
- Engineered time-series features including Vietnamese holidays, double-day indicators, cyclical calendar variables, lag features, and rolling-window statistics.
- Tuned and compared four machine learning models using MAE, RMSE, and R², with CatBoost achieving the best notebook experiment performance.
- Achieved MAE of 71.4K, RMSE of 99.7K, and R² of 0.9960 in the experimental evaluation period.
- Reduced MAE by 44.0% and RMSE by 48.7% compared with Random Forest, demonstrating the advantage of boosting models for daily e-commerce revenue forecasting.
- Translated model outputs into business implications for inventory allocation, promotion planning, and logistics capacity planning.
